## よく使う超・超・基本コマンド

まずは超基本コマンドです
普段から使われているものだと思いますが、
改めておさらいしましょう。

- 保存: `Command+S` / `Ctrl+S`
- 全選択: `Command+A` / `Ctrl+A`
- コピー: `Command+C` / `Ctrl+C`
- 貼り付け: `Command+V` / `Ctrl+V`
- 新規作成: `Command+N` / `Ctrl+N`

## 選択／移動を速くするコマンド

続いてテキスト内の
移動や選択に関わるコマンドです。

「ダブルクリック」
 → 単語単位の選択

「トリプルクリック」
 → 行単位の選択

`Command + ←/→` / `Home/End`
 → 行頭/行末へ移動

`Option+←/→` / `Ctrl+←/→`
 → 単語単位で移動

1つ1つは地味ですが、積み重なるとかなりの
作業効率UPにつながります。

## Google系サービス共通のコマンド

- リンク挿入: `Command+K` / `Ctrl+K`
- 直前操作の繰り返し: `Command+Y` / `Ctrl+Y`

個人的には直前操作の繰り返しが強力で、
スプレッドシートで「行削除」や「罫線再描画」
など、ショートカットに対応していない操作を
何度も繰り返せるのが非常に便利です。
