## APIとは？

API（Application Programming Interface）とは、サービス同士をつなぐ“窓口”という意味になるのですが、これだけだとよくわからないですよね。

簡単に言うと「既存サービスの便利な機能を借りられる仕組み」で、
自分で一から作れない機能を、他のサービスから貸してもらうことができます。

APIを使ってできることの一例
・天気情報：特定の地点の天気を取得
・株価・仮想通貨：リアルタイムの価格情報を取得
・Googleサービス：スプレッドシートの読み書き、カレンダー操作
・AI機能：チャットボットの回答生成

## 料金体系とAPIキー

APIには「無料で使える」ものと「使うたびに課金が発生する」ものがあります。

後者の課金が発生するものは誰がアクセスしているかを判別するために「APIキー」という鍵が使われ、
この鍵を使った回数に応じてその人の課金額が決定されるという仕組みとなります。

## APIキーの取り扱いに注意

APIキーが別の人に渡ってしまうと別の人が自分の鍵を持っている状態になるので、
勝手にサービスを使われた上にその分の料金があなたに請求されてしまうということが起こり得ます…！

特に開発初心者の場合は
・ログに出力
・GitHubに間違ってアップロード
・コードに直接記述
といったミスによってAPIが外部に漏れてしまうことがあるので、取り扱いには細心の注意を払いましょう！
