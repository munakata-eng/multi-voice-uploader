## プログラミング言語って？

「プログラミング言語」とは
人間がコンピュータを制御するために使う
プログラムを書くためのルールです。

プログラミング言語にはいくつもの種類があり
言語ごとに得意とする領域や書き方の決まり、
実行速度や開発方法が異なります。

## よく使われるプログラミング言語

### C
コンピュータの動作を細かく制御できる言語で
家電や電子機器の内部、OSといった
機械に近い部分を扱うことができます。

### Swift
Appleが開発した言語で、見た目が整った
アプリを作れるのが特徴です。
iOSのアプリを作るために使われます。

### Kotlin
主にAndroidアプリの開発で使われる言語で
短く書けて読みやすい特徴があります。
エラーが起きにくい仕組みが整っていて、
Googleが公式に採用しています。

### Python
AIや機械学習、データ分析、自動化ツールなど
幅広い分野で使われる言語です。
文法がやさしく参考情報も多いので
AIプログラミングと相性が良いです。

### JavaScript / TypeScript
JavaScriptはWebサイトの動きを作るための言語で
そこに"型"を付けて改良したのがTypeScriptです。

Node.jsという技術と組み合わせることで
Webサイトの見た目からサーバの裏側まで
幅広い範囲を作ることができるので、
とても人気がある言語となっています。


## プログラミング言語の選ばれ方

基本的には「作りたいもの」が決まると
それに合わせた言語が自動的に決まります。

AIプログラミングで開発するときも
AIが最適なものを自動で選んでくれるので、
裏側でこういった言語が使われてるんだな
程度で覚えておいていただければOKです。

Webサービス
→ JavaScript / TypeScript

コマンド上のツールやBot作成
→ Python

iPhoneやAndroidアプリ
→ Swift / Kotlin

## まとめ

プログラミング言語はたくさんの種類がありますが
「何を作るか」によって扱うものが決まります。

AIプログラミングでは、AIが自動的に判断して
最適な言語を選んでくれるので
扱う側はそこまで意識しなくてOKです！
