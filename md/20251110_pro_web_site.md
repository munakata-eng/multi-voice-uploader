# AIで作ったウェブサイトをプロっぽく見せる3つのコツ

こんにちは、むなかたです。

今回は、AIプログラミングで
ウェブサイトを作ったときに、
「なんかいまいちだな...」と
感じたときの改善方法について
お話ししていきます。

AIを使えば簡単にウェブサイトを
作ることができますが、
どうしても「AIで作った感」が
出てしまうことがありますよね。

今回は、その「AIっぽさ」を
一気に払拭できる3つのコツを
ご紹介します！

同じ内容は音声でも配信しているので、
ぜひお好きな方法でインプットしてくださいね！

## AIで作ったサイトの共通課題

まず、AIプログラミングで
サイトを作ると、どんな問題が
起きやすいのでしょうか。

### 似たり寄ったりな見た目になる

AIは学習してきたものの
**平均値**を取るため、
どうしても似たようなデザインに
なりがちです。

先日、メンバー限定のライブで
各AIツールを使ってサイトを
作り比べてみたのですが、
それぞれに特徴がありました。

### 各AIツールの特徴

**Gemini CLI**
- 見た目がシンプルすぎる傾向
- 枠線で囲っただけのグループ分け
- 単純すぎる構成になりやすい

**Codex**
- Geminiほどではないが、
  デザインが一発目ではいまいち
- やはりシンプルすぎる傾向

**Claude Code**
- グループ分けが比較的しっかりしている
- 文字の大きさや下線を活用
- 意味のある塊を分けてくれる
- ただし、絵文字を多用しがち
  （ロボット、ロケットなど）
- これがちょっとダサい印象に...

一発目の出力では、
どのAIもちょっと微妙なんです。

でも、これから紹介する3つのコツで
一気に改善できます！

## コツ1: Font Awesomeでアイコンを使う

まず一つ目は、
**アイコンを使用する**ことです。

### なぜアイコンが効果的？

1. **ぱっと見で内容が分かる**
   カテゴリーが視覚的に理解しやすい

2. **絵文字のダサさがない**
   Claude Codeの絵文字問題を解決

3. **プロフェッショナルな印象**
   一気に洗練された雰囲気に

### 具体的な指示方法

AIに以下のように指示します：

```
Font Awesomeのアイコンを使って、
このサイトの表示をもうちょっと
見やすくしてください。
```

**Font Awesome**は、
エンジニア界隈では非常に有名な
アイコンライブラリです。

この一言でかなり見やすくなります！

## コツ2: Googleフォントでフォントを変更

二つ目は、
**フォントを変更する**ことです。

### Googleフォントとは？

インターネット上から
フォントデータをダウンロードして、
どの端末でも同じフォントで
表示できる仕組みです。

### おすすめのフォント

**Noto Sans JP**
- 多くのサイトで使われている定番
- 読みやすく、プロフェッショナル

**Rounded M+**
- 丸っこくてかわいい印象
- 柔らかい雰囲気のサイトに最適

他にも「Googleフォント おすすめ」で
検索すると、たくさん紹介サイトが
見つかります。

### 具体的な指示方法

AIに以下のように指示します：

```
Googleフォントを使って、
Noto Sans JPのフォントを
このサイトに適用してください。
```

これだけで、サイトの印象が
がらりと変わります！

### 注意点：フォントは1サイト1つまで

フォントファイルは、
一文字一文字のデザインデータを
持っているため、
**ファイルサイズが大きくなりがち**です。

そのため、1サイトに
たくさんのフォントを入れるのは
おすすめしません。

**1サイト1つのフォント**が
ベストプラクティスです。

## コツ3: 素材サイトから画像を追加

三つ目は、
**画像やイラストを入れる**ことです。

これだけでも印象が
かなり変わります！

### おすすめの素材サイト

#### 1. Pixabay（実写写真向け）

- **URL**: https://pixabay.com/
- **特徴**: 無料で使える実写画像が豊富
- **用途**: ビジネス向け、真面目なサイト
- **効果**: プロっぽい印象になる

利用規約は事前に確認してくださいね。

#### 2. 素材やCNP（かわいい系向け）

- **特徴**: クリプト忍者パートナーズの
  キャラクターが使える
- **用途**: 砕けた感じ、かわいいサイト
- **効果**: 親しみやすい雰囲気に

### 具体的な使い方

1. 素材サイトから画像をダウンロード
2. VSCode上にドラッグ&ドロップ
3. AIに「この画像を使って表示してね」
   と指示

これで完成です！

### 2. Googleフォントでフォント変更


### 3. 素材サイトから画像を追加

からダウンロードして使用

この3つは**小手先のテクニック**ですが、
非常に効果的です！

## まとめ：AIっぽさを防ぐサイト作りのコツ

最後に、重要なポイントを
まとめます。

### AIで作ったサイトの課題
- 似たり寄ったりな見た目
- シンプルすぎるデザイン
- 絵文字の多用（Claude Codeの場合）

### 解決策：3つのコツ
1. **Font Awesome**でアイコン追加
   → 視覚的に分かりやすく、プロっぽく

2. **Googleフォント**でフォント変更
   → 印象をがらりと変える
   → ただし1サイト1つまで

3. **素材サイト**から画像追加
   → Pixabay（実写）
   → 素材やCNP（イラスト）
