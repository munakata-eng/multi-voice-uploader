## マークダウンとは何か？

マークダウンは、「記号だけで文章を装飾できる書き方」で、シンプルなテキストエディタでも
見出しや箇条書き、太字などの装飾を表現できるのが特徴です。


- **従来の装飾**：文字の一部を選択して太字や文字サイズ変更のボタンを押す
- **マークダウン**：記号のみで装飾の意味を記述する



### 見出し
```
# 大見出し
## 中見出し
### 小見出し
```

### リスト
```
- 項目1
- 項目2
- 項目3
```

### 太字・強調
```
**太字**
*斜体*
```

### コード表示
```
`コードの一部`
```

## なぜマークダウンが重要なのか？

### 1. AIとの相性が抜群

マークダウン最大の利点は、**AIとの相性が非常に良い**ことです。

AIはシンプルなテキストでのやり取りがメインなのですが、ただのテキストだけで装飾を表現することができます。
AIに「マークダウンで出力して」とお願いするだけで、構造化された文章を作ってくれるのでぜひ覚えておきましょう。

ちなみにマークダウンは単なるテキストなためコンテンツをGitやGitHubで管理することが可能で、
生成してもらった文章のバックアップや情報蓄積にも相性が良いです。


### 2. 対応サービスが豊富

主に開発者向けのサービスでは、マークダウンに対応したものが多く存在します。

マークダウンで書いた文章を変換してくれたりするものも多いので、
普段お使いのサービスが対応しているかをチェックしてみてもいいですね！

- **note**：マークダウンを貼り付けると自動で装飾が反映
- **VSCode**: 「プレビュー」ボタンから構造化された見た目をチェック可
- **Discord**：マークダウンでテキストを書くと装飾された文章を投稿できる


### 3. 文章加工がしやすい
マークダウンで書いた文章は、AIが加工するのも簡単です。

実はこのメルマガもマークダウンで事前に下書きを作っていて、
変換のプログラムによって見出しの部分の装飾を自動でつけてから送信しています笑

マークダウンでコンテンツのベースを作っておくことで、同じコンテンツを複数の媒体で再利用することも可能になるのです。
