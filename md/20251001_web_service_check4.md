# Webサービス公開時にチェックすべきの4つのポイント
こんにちは、むなかたです。

今日はサイトを公開する時に確認しておきたい
4つのポイントについて紹介していきます。

同じ内容は音声でも配信しているので、
ぜひお好きな方法でインプットしてくださいね！

## 1. サイトが正しく表示されているか

まずはサイトの表示チェックです。
PCだけでなくスマホでサイトを見る人も多いので
どちらの端末で見ても表示が崩れていないか
一度確認しておくのがオススメです。

画像やページが見つからないという問題も
サイト公開後によく起きがちなので、
一通りページを回遊して確認しておきましょう。

## 2. サイトの基本設定が整っているか

サイトの基本設定としてオススメの項目は
「Favicon」と「OGP」の2つです。

必須ではないですが、印象がガラッと変わるので
ぜひ設定しておきましょう。

### Favicon
ブラウザのタブに表示される小さなアイコン。
「favicon.ico」というファイルを作れば
設定できるのでサクッとやってしまいましょう。

### OGP
SNSにシェアしたときに表示されるサイト概要。
画像やタイトル、説明文が自動表示されるので
設定しておくと共有時の見栄えが段違いです。

## 3. 表示速度

サイトを表示したときの読み込み速度を
チェックするのも重要なポイントです。

特に画像・動画やフォントファイルは
サイズが大きくなりがちなので、
体感で表示がワンテンポ以上遅い場合は
大きいファイルの読み込みが行われていないか
一度確認してみましょう。

## 4. セキュリティと情報露出

こちらは少しステップアップした内容ですが
セキュリティの観点からのチェックも重要です。

・動作確認用のログ
・デバッグ用の機能
・仮で作ったテスト表示

など、不必要な情報が公開されていないか
確認しておくのがオススメです。
